When setting up a scheduler it is important to configure a time zone as it affects the time when a scheduler is run.

Timezone is configured on the connection level.

In the example below, a connection is configured with UTC timezone:

```
# yaml-language-server: $schema=https://cloud.dqo.ai/dqo-yaml-schema/ConnectionYaml-schema.json
apiVersion: dqo/v1
kind: source
spec:
  provider_type: bigquery
  bigquery:
    source_project_id: bigquery-public-data
    billing_project_id: dqo-ai-testing
    authentication_mode: google_application_credentials
    quota_project_id: dqo-ai-testing
  time_zone: UTC
  schedule:
    cron_expression: "*/1 * * * *"
    disabled: false
```
